* Lamprey Schema

This repository contains a minimal working example that is intended to
demonstrate how Pydantic could be used to validate the data contained
in a GLFC Lamprey data submission.  It contains basic validators for
both Gear and BioData tables.

Currently, the script fetches the data from the specified access
database (*.accdb) or excel template (*.xlsx). Any validation errors
are accumulated in an array that is reported at the end of the script.

This is a minimal, proof of concept.  Additional improvements include
updates to the validator, tests, more meaningful error reports, and a
better user interface (either as a desktop application or a web based
service).

** Installation

Create and activate python virtual environment:

#+begin_src bash

  mkvirtualenv lamprey_schema
  workon lamprey_schema

#+end_src

Install the required packages:

#+begin_src bash

  pip install -r requirements.txt

#+end_src


And then run the python script in a terminal:


#+begin_src bash

  python validate_data.py -s <YOUR-SOURCE-FILE>

#+end_src

Where <YOUR-SOURCE-FILE> is the path to your populated MSExcel or
MSAccess template.
